Ollama AI Chatbot

A local AI chatbot application built using Node.js, Express, and Ollama that runs an LLM directly on your machine.

This project demonstrates how to integrate a locally hosted Large Language Model (LLM) into a full-stack web application.

⸻

🚀 Features
	•	💬 Chat interface similar to modern AI assistants
	•	🧠 Runs a local LLM using Ollama
	•	⚡ Built with Node.js + Express
	•	📜 Conversation history support
	•	⌨️ Press Enter to send messages
	•	🤖 AI typing indicator
	•	🎨 Clean chat UI with HTML + CSS

⸻

🧠 Tech Stack

Frontend
	•	HTML
	•	CSS
	•	JavaScript

Backend
	•	Node.js
	•	Express

AI / LLM Runtime
	•	Ollama

Model Used
	•	TinyLlama (1B parameter LLM)

🏗️ Project Architecture

    User Interface (Browser)
        ↓
    Frontend (HTML / JS)
        ↓
    Express API Server
        ↓
    Ollama API
        ↓
    TinyLlama LLM
